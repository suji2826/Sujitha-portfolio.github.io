# 🌿 Sujitha Shanmugam | Portfolio Website

A personal portfolio website showcasing my skills, projects, and coding profiles — built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo

>https://suji2826.github.io/my-portfolio.github.io/


---

## 📌 Features

- **Responsive Design** — Works across desktop and mobile screens
- **Dark / Light Mode Toggle** — Smooth theme switching with a moon/sun icon
- **Animated Hero Section** — Fade-slide entrance animation
- **Skills Section** — Hover-interactive skill cards
- **Coding Profiles** — Direct links to LinkedIn, LeetCode, CodeChef, GFG, and GitHub
- **Projects Showcase** — Cards with tags and live demo links
- **Contact Section** — Phone and email details
- **Feedback Form** — Client-side form with success message
